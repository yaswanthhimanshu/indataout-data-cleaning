What inDataOut Does For You

🗑️
Remove Duplicates & Empty Columns
Automatically detects and removes duplicate rows and columns that are entirely empty or have a single repeated value across all rows.
🔢
Smart Missing Value Filling
Fills empty numeric cells with Mean, Median, or Zero. Fills empty text cells with the most frequent value (Mode) or the string "Unknown".
🏷️
Data Type Conversion
Auto-detects text columns that contain numbers and converts them. Manually override any column's type — Numeric, Datetime, String, Category, Boolean.
📉
Interactive Outlier Detection
IQR-based statistical outlier detection with a full preview table. You inspect and select exactly which rows to remove — no surprises, full control.
📊
Data Quality Score
Computes a 0–100 quality score before and after cleaning, based on missing values, duplicate rows, outliers, and constant columns.
🔒
Privacy-First Processing
Files live in isolated per-session folders on the server and are auto-deleted after download. Nothing is stored permanently.
